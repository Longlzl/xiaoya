# xiaoya
## 环境说明
使用Python3.12版本，
同时为了保证使用正确的websocket包，需要执行以下命令。
``` 
pip uninstall websocket
pip uninstall websockets 
pip install websocket-client
```
## 使用说明
### 安装谷歌的llm包 
```
pip install google-generativeai
``` 
### 安装星火的llm包 
```
pip install --upgrade spark_ai_python
``` 
### 安装gradio包 
```
pip install gradio
```
运行gradio_ui.py即可
